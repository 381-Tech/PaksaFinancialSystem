1. Check if Node.js is installed

Open a command prompt and run:

node --version

You should see a version number (e.g., v16.x.x or higher).

2. Install dependencies

In the project directory, run:

npm install

This will install all required packages.

3. Start the development server

Run the following command in the project directory:

npm run dev

You should see output indicating the server is running on port 3000.

4. Check for errors

Look for any error messages in the terminal where you ran npm run dev.

Common issues include:

  • Port 3000 already in use
  • Missing dependencies
  • Permission issues

5. Try alternative solutions

If you're still having issues, try these steps:

  1. Delete the node_modules folder and package-lock.json
  2. Run npm cache clean --force
  3. Run npm install again
  4. Try a different port by modifying vite.config.ts
